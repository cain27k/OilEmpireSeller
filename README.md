# Install Tesseract OCR

This application requires Tesseract OCR to detect the current sell price.

## Download Tesseract

Download here:

[https://github.com/UB-Mannheim/tesseract/wiki](https://github.com/UB-Mannheim/tesseract/releases/tag/v5.4.0.20240606)

Install using default settings.

Default install location:

```text id="7p1b9x"
C:\Program Files\Tesseract-OCR\tesseract.exe
```

If your installation location is different, update it inside the application Settings.

# First Time Setup

## Configure Price Region

* Click `Set Price Top Left`
* Hover the TOP LEFT corner of the Current Price box
* Wait 3 seconds

Then:

* Click `Set Price Bottom Right`
* Hover the BOTTOM RIGHT corner of the Current Price box
* Wait 3 seconds

---

## Configure Sell Button

* Click `Set Sell Button`
* Hover over the Sell button
* Wait 3 seconds

---

## Test OCR

Click:

```text id="u8t1pj"
Test Price Read
```

The app will save:

```text id="xw18ly"
price_debug.png
```

Make sure the image clearly shows the current sell price.
